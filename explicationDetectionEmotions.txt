****les caractéristiques *********

- couche de convolution : (extraction de caractéristiques d'images) appliquer les filtres pour extraire des caractéristique local 
(nb filtres,kernel_size, strides (optionel) ,padding(optionel ) ,activation(optionel),input_kernel)
dans tp on a choisi :Conv2D(64, (3, 3), activation='relu', input_shape=(48, 48, 1)))

(sigmoid => (val entre 0 et 1)  classification binaire plus adapté que tanh  (entre -1 et 1 ) 
( pour les couches cachés des neurones : on a Relu , Relu Leaky et Elu ) 
dans cet tp on a utilisée Relu (couche caché des neurones)  puisque il est plus simple

-BatchNormalization : Normaliser les activations pour stabiliser l'entraînement.


-pooling : 2 types: maxpooling ,averagePooling : reduire la dimension cartes caractéristique extraite
tout en conservant l'esssentiel 


-Dense : pour faire des predictions finales d'aprés les caractéristiques extraite de les couches précedent 
dense (nb neurone , activation ) 
(
on a utilisée softmax  car elle est classification multiclass( joie ,tristesse,surprise,etc)  )

-régularisation (drop out ,l2 regularization ) : pour éviter le surrapprentissage 
Dropout : Réduire le surapprentissage en désactivant certains neurones de manière aléatoire.


-flatten :transformer multiDimension en un seul vecteur 


- optimisation : adam plus famous 

-loss function :La fonction de coût mesure l'écart entre les prédictions du modèle et la vérité réelle
3 types :
Binary Cross-Entropy : Utilisée pour les tâches de classification binaire.
Categorical Cross-Entropy : Utilisée pour la classification multiclass(notre cas ).
Mean Squared Error (MSE) : Utilisée pour les tâches de régression.

---Répétition des étapes de convolution et de pooling plusieurs fois
 pour extraire des caractéristiques de plus en plus complexes.
